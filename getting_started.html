Getting Started¶

You need a device, this can be an actual physical device such as those listed on the LightNVM hardware page. Or it can be virtual device, such as one provided by qemu-nvme.

Next, you need operating system support, for Linux you can use the kernel sources available here:

  • git@github.com:OpenChannelSSD/linux.git
  • Specifically branch “liblnvm”

Lastly, you need liblightnvm itself, clone it from:

  • https://github.com/OpenChannelSSD/liblightnvm
  • Specifically branch “master”
